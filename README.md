# Autowall
Autowall is a very simple script to automatically download and set wallpapers from [Wallhaven.cc](https://wallhaven.cc/)

## Installation
Just clone and run ` sh install.sh `.

```bash
git clone https://github.com/begbaj/autowall
cd autowall
sh install.sh
```
